## Quick Start

### Web App (PWA)
1. Visit the deployed URL
2. Click "Install" to add to your device
3. Use online or offline!

### Desktop App
Download the standalone desktop application:
- **Windows**: Download and run `Tyrannosocial Setup.exe`
- **macOS**: Download and open `Tyrannosocial.dmg`
- **Linux**: Download and run `Tyrannosocial.AppImage`

Works completely offline after installation! See [ELECTRON_BUILD.md](./ELECTRON_BUILD.md) for building instructions.

## Development

```bash
# Install dependencies
npm install

# Run web development server
npm run dev

# Run desktop app in development mode
npm run electron:dev

# Build web version
npm run build

# Build desktop apps
npm run electron:build        # Current platform
npm run electron:build:win    # Windows
npm run electron:build:mac    # macOS
npm run electron:build:linux  # Linux
```
